## 📖 簡介

**SK Skills** 是 VIPthink 港澳 CC 團隊的 AI 教練技能庫，專為在線、少兒數學思維教育場景設計，服務港澳市場的課程顧問（CC）團隊。

包含销售话术、数据分析、教练系统、CRM 自动化等 AI 工具，幫助 CC 團隊提升業績、優化家長溝通、實現智能化團隊管理。

### 4️⃣ performance-report（業績彙報系統）

### 5️⃣ business-analysis（數據業務分析框架）
> 用渠道路徑 + 業務詞典做深度診斷分析

**核心指標：**
- DCP率（當日到課且成交 ÷ 日到課學員）
- 應到課率（到課數 ÷ 約課數）
- 到課轉化率
- 首撥間隔、課前2h跟進率
- T+0/T+1/T+2 撥打率

**輸出：** 飛書文檔格式的分析報告，用於周會診斷

