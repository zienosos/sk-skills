# 同事性格觀察 Skill

> 教練系統的觀察工具。從「說話方式」+「問問題方式」理解每個同事的性格。

---

## 決策樹（快速入口）

```
收到同事@提問
    │
    ├── 讀取senderLabel → 識別同事名字+組別
    │
    ├── 分類問題類型
    │       ├── 政策（先付一半/月付/逐堂買）→ 🔴立刻記錄⚠️
    │       ├── 話術/異議 → 記錄為「技能應用」
    │       ├── 積極回饋 → 記錄為「✅積極信號」
    │       └── 教練廣播沉默 → 記錄為「🟡沉默信號」
    │
    ├── 寫入colleague_conversations.md（頂部插入）
    │
    └── 回覆（先記錄再回覆）
```

---

## ⚠️ 鐵律

**每次在02組群/03組群回覆同事@提問，必須：**
1. 先讀取發送者名字（senderLabel）
2. 先寫入對話記錄文件
3. 再思考回覆內容

---

## 核心功能

| 功能 | 說明 |
|------|------|
| **對話記錄** | 每次同事@我的提問，完整記錄 |
| **說話風格識別** | 從問問題方式判斷性格 |
| **高頻問題追蹤** | 統計每人的問題分佈 |
| **問題圖譜** | 每月畫出問題分佈，識別技能卡點 |
| **教練切入點** | 根據性格+問題給切入建議 |

---

## 對話記錄（即時追加）

打開 `coaching/colleague_conversations.md`，在頂部插入：

```markdown
## YYYY-MM-DD HH:MM

**同事：** [名字]（[組別]）
**問題類型：** [政策/話術/異議/數據/教練/積極回饋]
**問題內容：** [同事問的原文，粵語保留，一字不漏]
**問題翻譯：** [普通話大意]
**我的回覆摘要：** [我點答的核心內容]
**教練意義：** [說話風格/性格信號/教練切入點]
```

## 說話風格識別（核心維度）

| 風格 | 特徵 | 問問題方式 | 代表同事 |
|------|------|-----------|---------|
| 務實型 | 直接問操作問題，唔拖泥帶水 | 一次性問多個，具體明確 | 陳振成 |
| 行動派 | 主動追問，想要完整解決方案 | 一次性傾曬，要完整話術 | 詹敏玲 |
| 審慎型 | 願意深度跟進，節奏慢但扎實 | 圍繞同一個案深入 | 余景馳 |
| 概念清晰型 | 先確認定義，有條理有邏輯 | 先問定義，再問操作 | 羅家欣 |
| 穩陣型 | 要具體方向，踏實務實 | 想要step by step | 陳迪鑫 |
| 直接犀利型 | 語氣強硬，唔留情面 | 少問題，但鬧人好快 | 楊禮萍 |
| 沉默觀察型 | 少說話，被動回應 | 真正需要才問 | 潘海麗 |
| 沉默落後型 | 完全無回應，消極被動 | （幾乎唔問） | 劉金雅 |

## 高頻問題追蹤（問題圖譜）

每次記錄時，在 `coaching/colleague_profiles.md` 更新該同事的問題統計：

```markdown
### [名字]（[組別]）

**說話風格：** [描述]
**問題圖譜（本/月）：**
- 價格異議：2次
- 政策邊界：1次
- 售後服務：1次

**性格關鍵詞：** [2-3個關鍵詞]
```

## 常見錯誤

| 錯誤 | 正確做法 |
|------|---------|
| 只記錄問題類型，唔記錄問題原文 | **一字不漏**記錄問題內容 |
| 先回覆再記錄 | 先記錄再回覆 |
| 忘記寫教練意義 | 每條都要寫 |
| 問題內容翻譯成普通話 | 保留粵語原文 |
