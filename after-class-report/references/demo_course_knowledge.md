# Demo 課程知識庫

來源：飛書文件夾「益智demo課程介紹長圖」  
更新日期：2026-05-24

用途：生成港澳家長版課後報告時，根據 demo 課名稱匹配數學知識點、能力解讀和家長可理解表達。主程序優先讀取 `demo_course_knowledge.json`。

## S1

- 光影魔術師：配對問題，含功能配對、特徵配對、顏色配對、形狀配對、數字/湊整配對。重點能力：觀察比較、一一對應、分類意識、生活化推理。
- 海底派對：圖形、顏色、大小和分類。重點能力：基礎觀察、語言表達、按特徵分類、規則理解。
- 樂迪的一天：先後順序、時間順序、事件排序。重點能力：順序表達、因果理解、任務規劃。
- 最棒的演出：位置對應、簡單排序、規則匹配。重點能力：聽指令、按規則行動、專注維持。

## S2

- 歡樂博物館：多角度分類、共同特徵、圖形比較。重點能力：多維觀察、分類歸納、表達理由。
- 歡樂舞會：AB規律、顏色/形狀規律、隊列排序。重點能力：規律觀察、預測下一步、視覺專注。
- 皮皮虎的禮物：數量比較、多少判斷、一一對應、公平分配。重點能力：數感建立、比較語言、生活應用。

## S3-S5

- 魔法集市奇遇記：數量守恆、條件篩選、簡單買賣、分類比較。
- 新智闖懸浮島：方向辨認、路徑規劃、位置關係、空間想像。
- 魔法水源：條件推理、排除法、順序分析、原因結果。
- 逃離暴風雨：路徑選擇、空間方位、條件限制、策略比較。
- 外星來客小柴柴：圖形特徵、條件匹配、分類判斷、規律觀察。
- 超時空任務：時間順序、條件整合、邏輯推理、多步任務。
- 廚神挑戰賽：數量分配、倍數雛形、步驟排序、條件比較。
- 馬上前去救援：路徑規劃、條件限制、最短/合適路線、策略判斷。

## S6-S8

- 世紀末大偵探：線索推理、排除法、真假判斷、條件組合。
- 尋找芳香草：特徵分類、條件篩選、集合交集、排除推理。
- 決戰銀河金杯星：綜合推理、策略比較、數量關係、規則運用。
- 世紀末大偵探 S7：多條件推理、邏輯鏈、排除法、證據驗證。
- 摩天樓數獨：數獨規則、行列約束、空間高度、排除推理。
